ปลอดภัยและเป็นไปตามพระราชบัญญัติคุ้มครองข้อมูลส่วนบุคคล (PDPA) ของไทย · ข้อมูลถูกเก็บใน data center ที่มาตรฐาน ISO 27001 · เข้ารหัสทั้ง in-transit และ at-rest · ลูกค้าสามารถขอลบข้อมูลได้เมื่อต้องการ · เรามี Data Processing Agreement สำหรับลูกค้า บิสซิเนส แพ็ค

ขั้นตอนการตั้งค่าครั้งแรกใช้เวลานานไหม?

ปกติประมาณ 10-15 นาที · 5 ขั้นง่าย ๆ: ข้อมูลบริษัท → โลโก้ → เพิ่มสินค้าแรก (import จาก PDF ได้) → เชิญทีม → เชื่อมต่อ · ผู้ช่วยจะนำทางทุกขั้น · ถ้าติดขัด นัด การตั้งค่าเริ่มต้น 1-on-1 กับทีมเราฟรีได้ (paid แพ็ค)

เหมาะกับธุรกิจประเภทไหน?

ตอนนี้ focus ที่ เฟอร์นิเจอร์สำนักงาน · อุปกรณ์ออฟฟิศ · เครื่องใช้ในบ้าน เพราะ เทมเพลต + แคตตาล็อก structure ออกแบบมาให้เหมาะ · แต่ใช้กับธุรกิจอื่นที่ขายผ่านไลน์ และต้องส่งใบเสนอราคาได้ทุกประเภท · ถ้าไม่แน่ใจ ลองคุยกับเราก่อน

ผู้ช่วยใช้เอไอ อะไร · ทำอะไรบ้าง?

ใช้ Claude (จาก Anthropic) เป็น ตัวเสริม ให้เซลส์ทำงานเร็วขึ้น · ช่วย ดึงข้อมูล ข้อมูลจาก เอกสารสเปก ตอน ใส่ข้อมูล แคตตาล็อก · จับคู่สินค้าใน แชทไลน์ · เสนอราคาเริ่มต้นจากค่าเฉลี่ย · เซลส์ตรวจและตัดสินเองทุกครั้ง · เอไอไม่ส่งใบเสนอเอง · ไม่ตอบลูกค้าแทน · เป็น helpful assistant ไม่ใช่ autopilot

ผลตอบแทนการลงทุนเป็นอย่างไร?

การลงทุนต่อเดือนเท่ากับเงินเดือนพนักงานครึ่งวัน · ถ้าระบบช่วยให้ทีมเหลือเวลาไปคุยลูกค้าใหม่ · หรือกู้ดีลที่เคยตกหาย 1 ดีลที่มูลค่า 200,000+ บาท · คืนทุนทันทีในเดือนแรก

ถ้าเลิกใช้ ข้อมูลทำอย่างไร?

ดาวน์โหลดข้อมูลทั้งหมดเป็น CSV / PDF ได้ก่อนเลิกใช้ · ลูกค้า · ดีล · ใบเสนอ · ประวัติแชต — ทั้งหมด · เก็บไว้ใน ไลน์ โอเอ ก็ยังเข้าถึงได้เหมือนเดิม · ไม่มี ผู้ขายระบบ lock-in

มี API ให้เชื่อมต่อกับระบบบัญชี / ERP ของเราไหม?

บิสซิเนส แพ็ค มี API เต็มที่สำหรับ ส่งออก ใบเสนอ · ลูกค้า · ดีล ไปยังระบบบัญชี (Express, FlowAccount, PEAK, Xero) · ใน แพ็คโปร ส่งออก ผ่าน CSV ได้ · ถ้าใช้ ERP ที่ specific มาก ๆ ทีมเรา customize ให้ได้
