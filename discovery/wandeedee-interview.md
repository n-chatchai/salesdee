# คำถามสัมภาษณ์ — วัน.ดี.ดี. (anchor customer / dogfooding)

> เป้าหมาย: เก็บ requirement จริงจากธุรกิจเฟอร์นิเจอร์ที่จะเป็นลูกค้ารายแรก เพื่อกำหนด MVP และเช็กว่าปัญหานี้ generalize ไปร้านเฟอร์นิเจอร์อื่นได้ไหม
> วิธีใช้: ถามแบบสนทนา ไม่ต้องเรียงตามนี้เป๊ะ; ตรงไหนได้คำตอบสั้น ๆ ให้ถาม "ยกตัวอย่างเคสล่าสุด" ต่อ
> 🔴 = สำคัญต่อการตัดสินใจ MVP / 📎 = ต้องขอเอกสารตัวอย่าง

---

## A. ภาพรวมธุรกิจ & ทีม
1. ขายอะไรบ้าง สัดส่วนรายได้แต่ละหมวด (โต๊ะ/เก้าอี้/ตู้เซฟ/ล็อกเกอร์/ตู้รางเลื่อน/ชุดประชุม/โซฟา) — และมี **งานสั่งทำ/บิวท์อิน** ด้วยไหม หรือขายของสำเร็จรูปล้วน? 🔴
2. ลูกค้าหลักเป็นใคร (เอกชน/ราชการ/โรงแรม/โรงเรียน/ผู้รับเหมา/end user) สัดส่วนคร่าว ๆ?
3. มีพนักงานกี่คน ใครทำอะไร — เซลส์กี่คน, แอดมิน/ประสานงาน, บัญชี, ทีมส่ง-ติดตั้ง (ทำเองหรือจ้าง)?
4. ออเดอร์เฉลี่ยมูลค่าเท่าไหร่? ออกใบเสนอราคากี่ใบ/เดือน? ปิดได้ประมาณกี่ %?
5. มีกี่สาขา/โกดัง/โชว์รูม?
6. มีสต็อกสินค้าของตัวเองไหม หรือสั่งจากซัพพลายเออร์/โรงงานเมื่อมีออเดอร์? lead time จากซัพพลายเออร์เท่าไหร่?

## B. ลูกค้าเข้ามาทางไหน (Lead)
7. ลูกค้าใหม่เข้ามาทางช่องทางไหนบ้าง สัดส่วน? (LINE OA / เพจ FB / โทร / เว็บฟอร์ม / อีเมล / walk-in โชว์รูม / ออกบูธ / e-bidding ภาครัฐ / แนะนำต่อ) 🔴
8. มี **LINE OA** ของบริษัทไหม ใครดูแล ตอบเร็วแค่ไหน เก็บประวัติแชตยังไง? 🔴
9. ตอนนี้ลูกค้า/ผู้ติดต่อใหม่ "เก็บ" ไว้ที่ไหน — สมุด / Excel / อยู่ในหัวเซลส์ / ไม่ได้เก็บ?
10. ใครเป็นคนรับลูกค้าเข้ามาก่อน แล้วส่งต่อให้เซลส์ยังไง? เคยมีลูกค้าหลุด/ตกหล่นไหม?
11. มีงานราชการ / e-bidding ไหม? กระบวนการต่างจากเอกชนยังไง (TOR, ราคากลาง, เอกสารคุณสมบัติ, ขึ้นทะเบียนผู้ค้า)?

## C. กระบวนการขาย & ใบเสนอราคา
12. ตั้งแต่ลูกค้าทักมา จนปิดการขาย มีขั้นตอนอะไรบ้าง? โดยเฉลี่ยใช้เวลานานแค่ไหน?
13. ตอนนี้ทำใบเสนอราคาด้วยอะไร (Excel/Word/โปรแกรมสำเร็จรูป)? ใช้เวลากี่นาที/ใบ? ใครทำ? 🔴 📎
14. ใบเสนอราคาหน้าตาเป็นยังไง — มี **รูปสินค้า** ในใบไหม? **จัดกลุ่มตามห้อง/โซน** ไหม? ใส่ **ขนาด W×D×H / วัสดุ / สี** ไหม? 🔴 📎
15. แก้ราคา/แก้สเปกบ่อยแค่ไหนต่อหนึ่งดีล? เก็บเวอร์ชันยังไง? เคยส่งผิดเวอร์ชันให้ลูกค้าไหม?
16. ส่งใบเสนอราคาให้ลูกค้าทางไหน (LINE / อีเมล / พิมพ์ / PDF)? ลูกค้าตอบรับด้วยอะไร (เซ็น PO / แชต / วาจา)?
17. มีการอนุมัติส่วนลดภายในไหม? ใครอนุมัติได้ถึงกี่ %?
18. หลังส่งใบเสนอราคาแล้ว ตามลูกค้ายังไง? ตามบ่อยแค่ไหน? เคยลืมตามแล้วเสียงานไหม? 🔴
19. คู่แข่งหลักคือใคร? แพ้งานเพราะอะไรบ่อยสุด (ราคา / ช้า / แบรนด์ / lead time / สเปกไม่ตรง)?
20. ภาษาเอกสาร — ไทยล้วน หรือมีอังกฤษด้วย (ลูกค้าต่างชาติ/บริษัทข้ามชาติ)? มีสกุลเงินอื่นไหม?

## D. สินค้า / แคตตาล็อก / ราคา
21. มีแคตตาล็อกสินค้าเป็นไฟล์ไหม (Excel / PDF / รูป)? ประมาณกี่รายการ? มีรูปครบทุกตัวไหม? 📎
22. สินค้ามีหลายขนาด/หลายสี/หลายวัสดุต่อรุ่นไหม (variant)? คิดราคาแยกยังไง?
23. ราคาเปลี่ยนบ่อยไหม? มีราคาตามลูกค้า/ตามปริมาณ (ขายส่ง vs ปลีก) ไหม? 📎 (ขอ price list)
24. ขายเป็น "ชุด" (เช่น ชุดประชุม 12 ที่นั่ง = โต๊ะ + เก้าอี้ ×12) ไหม? คิดราคายังไง?
25. มีออปชันเสริมต่อรายการไหม (ไฟ LED ใต้ตู้ / อัปเกรดเบาะหนัง / เพิ่มลิ้นชัก)?
26. รู้ต้นทุน/กำไรต่อรายการไหม? อยากเห็น **margin** ตอนทำใบเสนอราคาไหม (ให้เฉพาะเจ้าของ/ผู้จัดการเห็น)?
27. ถ้ามีงานสั่งทำ/บิวท์อิน — คิดราคายังไง (ต่อ ตร.ม. / ต่อชิ้น / ตามวัสดุ-BOM)? ขอราคาจากซัพพลายเออร์วัสดุก่อนไหม?

## E. มัดจำ / การชำระเงิน / วางบิล
28. เก็บเงินยังไง — มัดจำกี่ %? ก่อนติดตั้งเก็บอีกไหม? งานเสร็จเก็บส่วนที่เหลือ? เงื่อนไขมาตรฐานคืออะไร? 🔴
29. ลูกค้าองค์กรขอเครดิตกี่วัน (30/60/90)? ต้องวางบิลตามรอบของลูกค้าไหม?
30. มีงานที่ลูกค้าหัก **เงินประกันผลงาน (retention)** ไว้ % ไหม? คืนเมื่อไหร่?
31. ลูกค้าหัก **ภาษี ณ ที่จ่าย** ไหม? กี่ % (1% ค่าขนส่ง / 3% บริการ-ติดตั้ง)? ส่งหนังสือรับรองฯ ยังไง?
32. ลูกค้าค้างชำระเป็นปัญหาไหม? ตอนนี้ตามหนี้ยังไง?

## F. ส่งของ / ติดตั้ง / รับประกัน / หลังการขาย
33. lead time ตั้งแต่ยืนยันออเดอร์ (รับมัดจำ) ถึงส่งของ/ติดตั้ง โดยเฉลี่ยกี่วัน?
34. ติดตั้งเองหรือจ้าง? ใครจัดคิวงานติดตั้ง? ลูกค้าเซ็นรับงานยังไง (กระดาษ / ถ่ายรูป / ไม่มี)?
35. รับประกันกี่ปี (เห็นในเว็บว่า 1 ปี)? เก็บข้อมูลรับประกันยังไง? ลูกค้าเคลมมาทางไหน? ใครจัดการ? 🔴
36. มีงานบริการหลังการขาย / ซ่อม / ขายอะไหล่ไหม?
37. ลูกค้ากลับมาซื้อซ้ำเยอะไหม? มีการติดตามลูกค้าเก่า / ส่งโปรโมชันไหม?

## G. บัญชี / ภาษี / ระบบที่ใช้อยู่
38. ตอนนี้ใช้โปรแกรม/เครื่องมืออะไรอยู่บ้าง (บัญชี / ขาย / Excel / LINE / Google Sheet)? อะไรเวิร์ก อะไรไม่เวิร์ก? 🔴
39. ออกใบกำกับภาษี / ใบเสร็จ ด้วยอะไร? ใครออก? 📎 (ขอตัวอย่าง)
40. ใช้สำนักงานบัญชีภายนอกไหม? เขาต้องการข้อมูล/รายงานแบบไหน ส่งให้เขายังไง?
41. จด VAT ไหม? มีกี่สาขาในระบบภาษี (สำนักงานใหญ่/สาขา)?

## H. เป้าหมาย & ความคาดหวังจากระบบใหม่
42. ถ้ามีระบบใหม่ "อยากให้แก้ปัญหาอะไรเป็นอันดับแรก" — เรียง **top 3 pain**? 🔴
43. ใครจะเป็นคนใช้จริง? ใช้บนคอมหรือมือถือเป็นหลัก? เซลส์ใช้หน้างาน/นอกออฟฟิศไหม?
44. ยอมจ่ายค่าระบบประมาณเดือนละเท่าไหร่? (เทียบกับที่จ่ายอยู่ตอนนี้ ถ้ามี)
45. ขอ **ตัวอย่างจริง**: ใบเสนอราคา 2-3 ใบ (ทั้งงานเล็ก-ใหญ่), ใบกำกับภาษี/ใบเสร็จ, แคตตาล็อก, price list, ใบส่งของ/ใบรับประกัน (ถ้ามี) 🔴 📎
46. คิดว่าร้านเฟอร์นิเจอร์/ดีลเลอร์รายอื่นเจอปัญหาเดียวกันไหม? พอจะแนะนำใครให้ลองใช้ได้บ้าง (เผื่อเป็นลูกค้า beta)?

## เอกสารที่ต้องขอกลับมา (checklist)
- [ ] ใบเสนอราคาตัวอย่าง 2-3 ใบ (งานสำเร็จรูป + งานที่มีหลายห้อง/ติดตั้ง)
- [ ] ใบกำกับภาษี / ใบเสร็จรับเงิน ตัวอย่าง
- [ ] แคตตาล็อกสินค้า + price list (Excel ได้ยิ่งดี)
- [ ] ใบส่งของ / ใบตรวจรับงาน / ใบรับประกัน (ถ้ามี)
- [ ] ตัวอย่างแชต LINE กับลูกค้า 1-2 เคส (ดู flow จริง)
- [ ] เงื่อนไขการชำระเงิน/มัดจำ มาตรฐานที่ใช้

## สิ่งที่ต้องสรุปหลังคุยเสร็จ
- ขั้นตอนการขายจริง (เป็น flow / pipeline stages ที่ตรงกับเขา)
- ขอบเขต MVP ที่เขายอมใช้แทนของเดิมได้ (โดยเฉพาะ: ต้องมีใบกำกับภาษี/ใบเสร็จในเฟส 1 ไหม)
- ฟิลด์/หน้าตาใบเสนอราคาที่เขาต้องการ (เทียบกับ §4.7 ใน REQUIREMENTS.md)
- ความถี่/มูลค่า → ใช้ประเมิน scale และ pricing
- ปัญหา top 3 → ใช้จัดลำดับงานเฟส 1
